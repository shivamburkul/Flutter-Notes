## Introduction to Flutter

### What is Flutter?
Flutter is a free framework made by Google.
It is used to create apps for **Android, iOS, Web, and Desktop** using **one codebase**.
This means you don’t have to write separate code for each platform.

---

### Why Flutter is popular
- Write code once → works on multiple platforms
- Very fast development with **Hot Reload**
- Can make beautiful and smooth apps
- Many ready-to-use widgets
- Strong community and support

---

### What language does Flutter use?
Flutter uses **Dart**.
Dart is used for:
- Designing UI
- Writing app logic (what happens when user clicks, types, etc.)
- Managing state
- Calling APIs

Flutter does **not** need XML or HTML like native Android apps.

---

### What is Dart?
Dart is an easy-to-learn, object-oriented programming language by Google.
- Has **Null Safety**
- Works with Flutter to create apps
- Controls both how the app looks and behaves

---

### Flutter Architecture (Simple view)
Flutter has 3 main layers:

1. **Framework (Dart layer)**  
   - Widgets, Layout, Animation, Rendering

2. **Engine (C++ layer)**  
   - Graphics, Rendering using Skia

3. **Platform**  
   - Android, iOS, Web, Desktop

---

### Widgets
Everything in Flutter is a **widget**:
- Text, Button, Image, Screen, App
- Layouts like Row, Column, Stack
- Even padding, margin, alignment

Widgets are **building blocks** of a Flutter app.

---

### Two main types of widgets
1. **StatelessWidget**
   - Static UI, does not change
2. **StatefulWidget**
   - Dynamic UI, can change

---

### Hot Reload
- Lets you **see changes instantly**  
- Does not restart the app  
- Saves time while developing  

---

### Flutter vs Native Android (Easy comparison)
| Feature | Flutter | Native Android |
|--------|--------|----------------|
| Codebase | One for all platforms | Separate for Android & iOS |
| Language | Dart | Kotlin / Java |
| UI | Widgets | XML layouts |
| Development speed | Fast | Slower |

---

### Why Flutter is good for career
- High demand in startups and product companies
- Cross-platform skills = more jobs
- Can reach high-paying jobs if combined with backend skills

---

### Summary
- Flutter = framework for apps by Google  
- Uses Dart language  
- Everything is a widget  
- Hot Reload = faster development  
- One codebase for multiple platforms  
- Good career option if you learn properly
